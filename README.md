PDF Research Buddy

An AI-powered research assistant that lets you upload research papers, process them, and chat with them interactively. 
Built with Gradio and Retrieval-Augmented Generation (RAG), 
the tool extracts relevant sections from PDFs and provides clear, contextual answers to user questions.
The goal of this project is to make reading and understanding academic papers faster and more interactive.
